## 第 0 部分：问题 setup（你需要的全部定义）

考虑一个用于 unsupervised environment design (UED) 的极小极大问题。记号：

- `x ∈ X ⊆ ℝ^{d}`：student 策略参数（min 变量），对 x **非凸**。
- `Λ = {l_1,…,l_n}`：n 个候选 level（环境），固定有限。
- `y ∈ Δ_n`：adversary 在 n 个 level 上的课程分布（max 变量），`Δ_n` 为概率单纯形。
- `s(x) ∈ ℝ^n`：**score 向量**，分量 `s_i` 为 level `l_i` 的得分。`s` 仅通过线性项 `yᵀs` 进入目标。
- `H(y) = −Σ_i y_i log y_i`：Shannon 熵（强凹）。
- `α > 0`：熵正则系数。

**正则化博弈目标**：
```
f_α(x,y) = yᵀ s(x) + α H(y)
min_{x∈X} max_{y∈Δ_n} f_α(x,y)
Φ_α(x) := max_{y∈Δ_n} f_α(x,y)        （外层目标）
Φ_0(x) := max_{y∈Δ_n} yᵀ s(x)         （未正则化目标）
```

**关键结构**：strong-concavity 来自 `αH(y)`，**不来自** s 的任何性质（s 只线性进入）。故对**任意有界 score** `‖s‖_∞ ≤ B_s`，`f_α(x,·)` 都是 α-强凹。这是本方案的支点——它让我们能把 zero-sum 的 regret score 换成**异质 score**（learnability、coverage 等非 regret 信号）而保留收敛结构。

**假设**（A1 X 紧/coercive；A2 `‖s‖_∞≤B_s`；A3 `s(x)` 对 x 是 `L_s`-Lipschitz、`ℓ_s`-smooth；A4 随机梯度无偏、方差 ≤σ²；A5 s 仅经 `yᵀs` 线性进入；A6 两时间尺度步长 η_x/η_y→0）。

**算法**（你前两轮分析的对象）：随机两时间尺度，内层对 y 用 **KL-mirror ascent（指数梯度）**，外层对 x 用 SGD：
```
内层： y_{t+1} = argmax_{y∈Δ_n} { η_y⟨ŝ(x_t), y⟩ − η_y α H⁻(y) − D_KL(y‖y_t) }
外层： x_{t+1} = x_t − η_x ∇̂_x f_α(x_t, y_{t+1})
```

## 第 1 部分：你前两轮已经证明的结论（你自己的原话，作为本轮出发点）

- **定理 2（偏差）**：`0 ≤ Φ_α(x) − Φ_0(x) ≤ α log n`（Nesterov 熵平滑）。✅
- **G3（你在之前的推导中给出的总复杂度）**：
  ```
  K = O( 1/(αε²) + n·e^{2B_s/α}/(α³ε³) )
  ```
  你的论证：固定 α 时对 ε 多项式收敛成立；α→0（代入 α=O(ε_target/log n)）则对 1/ε_target **指数爆炸**。你指出指数来源是 **dual-to-primal drift（对偶-原空间映射漂移）**，其常数 `1/ν = n·e^{2B_s/α}`，`ν` 是 `y*=softmax(s/α)` 的**最小分量**。据此你建议 claim 改为「固定 α 收敛到 Φ_α」。

**本轮挑战的不是上面任何一条的正确性，而是 G3 那个指数项是否本可避免。**

具体地，你的 `e^{2B_s/α}` 来自跨步漂移项的这一步放缩（在你此前对 G3 两时间尺度耦合的推导中，处理「外层 x 更新导致内层最优点 y* 漂移」的那一步）：
```
D_h(y*(x_{t+1}) ‖ y*(x_t)) ≤ (1/ν)·‖y*(x_{t+1}) − y*(x_t)‖₁²,   ν = (1/n)e^{−2B_s/α}
```
即：先把漂移用 ℓ₁ 范数平方 bound，再用 `1/ν` 把 ℓ₁ 转回 Bregman 散度 `D_h`。

## 第 2 部分：新证据（实锤）——一篇**同目标函数**的已发表论文拿到了**多项式** 1/α 率

独立文献核查（2026，多源 3-vote 对抗验证，高置信）发现：

**Monette et al. 2025（arXiv:2505.20659，"An Optimisation Framework for Unsupervised Environment Design"，下称 NCC）：**

- **处理完全相同的目标函数**：`min_x max_{y∈Y} f(x,y) = yᵀs(x) + αH(y)`，`H(y)=−yᵀlog y`，对 x 非凸、对 y 熵-强凹。其 **Proposition 1**：`∇²_y f = diag(−α/y) ⪯ −αI`（与本工作 Hessian **完全一致**）。
- **它的率是多项式 in (1/α)，无任何 `e^{·/α}`**：
  ```
  K_NCC = O( Δℓ³/(α²ε²) + ℓ³σ²/(αε⁴) )      —— 多项式 in κ=ℓ/α
  ```
- **它如何避免边界爆炸**：**不走 mirror**。把 y 约束在 **δ-截断单纯形** `Δ_δ={y∈Δ_n | y_i≥δ}`，用**欧氏投影** `P_Y`（更新 `y_{t+1}=P_Y(y_t+η_y·Ĝ)`），然后**直接套 Lin-Jin-Jordan 2020 Thm 4.5**（D≤√2 为 Y 直径，κ=L/ρ）。

**对比的尖锐之处：**

| | NCC（已发表） | 你的 mirror 路线 |
|---|---|---|
| 目标函数 | `yᵀs+αH(y)` | **完全相同** |
| Hessian | `diag(−α/y)⪯−αI` | **完全相同** |
| 内层几何 | δ-截断单纯形 + 欧氏投影 | KL/Bregman mirror，**不截断** |
| 率 in 1/α | **多项式** `O(1/α²)` | **指数** `e^{2B_s/α}` |

**核心质疑**：同一个目标函数、同一个 Hessian，存在一条已发表的**多项式率**路线。那么你的指数率**几乎不可能是问题的内禀难度**，而更像是「mirror 不截断」这条路线 + 你那套漂移记账（`1/ν` 转换）的产物。
