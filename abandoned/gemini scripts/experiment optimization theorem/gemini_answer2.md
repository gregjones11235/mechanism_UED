### 0. 先证伪结论（D1–D4 裁决）

- **D1（L1 两个单调分量之差不单峰）**：**不塌（L1 成立）**。虽然 $P(\alpha)$ 是两个单调反向的量之差，但由于偏差项 $\alpha \log n$ 是关于 $\alpha$ 的线性项（或更紧的凸函数），而优化残差 $\epsilon_{\text{opt}}(\alpha)$ 是关于 $\alpha$ 的负次幂凸函数，**两凸函数之和仍为严格凸函数**。其导数 $P'(\alpha)$ 严格单调递增，在 $(0, \infty)$ 只能变号一次，故必为严格单谷（单峰的最小化等价），不会有多个驻点或平台。
- **D2（L1→L2 的桥假设其实是假的）**：**塌（L2 降级为工作假设）**。在 UED 的非凸神经网络设定下，优化层面的 $P(\alpha)$ 极小化意味着在特定候选集 $\Lambda$ 上达到了极小的 Minimax 偏差和优化残差。但在分布外或真实任务测试中，完美拟合 Minimax 目标往往导致对特定难度 level 的过拟合（Overfitting to the curriculum）。因此“更小的 $P(\alpha)$ 必导向更好的真实泛化 $G(\alpha)$”存在直接反例，桥假设失效，不能作为定理。
- **D3（峰位置不可解析估 / 偏离中点）**：**塌（几何中点法不可行）**。由于 $P(\alpha)$ 的两项系数分别取决于 $\log n$ 和预算 $K_{\text{budget}}$，其极小值点 $\alpha^*$ 的解析位置完全由这二者的比例决定，与区间的几何中点 $\alpha_{\text{geo}} = \sqrt{\alpha_{\text{min}}\alpha_{\text{max}}}$ **没有任何代数上的必然联系**。强行取 $\alpha_{\text{geo}}$ 极大概率会错过真正的最优点。
- **D4（已独立搜索 D1–D4 之外的证伪路径）**：**结果为未塌（但需修正细节以保严谨）**。
    1. **检查 $c$ 的 $\alpha$ 独立性 (见禁糊清单c)**：由 B4，若 $\Delta_\Phi = \Phi_\alpha(x_0) - \Phi_\alpha(x^*)$ 包含于复杂度常数中，由于 $\Phi_\alpha \sim \alpha \log n$，其实 $c$ 会微弱依赖于 $\alpha$（如 $\epsilon_{\text{opt}} \propto \alpha^{-1/2}$ 而非绝对的 $\alpha^{-3/4}$）。但这仅改变负幂次的具体数值（仍为 $p>0$ 的 $\alpha^{-p}$），不改变其严格凸性（因为 $(\alpha^{-p})'' > 0$ 恒成立）。
    2. **检查 instance-dependent 偏差界的凸性 (见禁糊清单e)**：若将 $\alpha \log n$ 替换为紧界 $B(\alpha) = \alpha \log(1+(n-1)e^{-\Delta/\alpha})$。令 $g(y) = \log(1+(n-1)e^{-y})$，其二阶导 $g''(y) > 0$，故 $g$ 是严格凸函数。$B(\alpha) = \alpha g(\Delta/\alpha)$ 正是 $g$ 的透视函数（Perspective function）。由于透视函数保凸性，紧界 $B(\alpha)$ 依然严格凸。无论用哪种界，$P(\alpha)$ 严格单谷性质牢不可破。

### 1. L1 证明（优化层：可达精度单峰）

**定义与求解**：
可达精度泛函（基于 B2 和 B4）：
$P(\alpha) = A(\alpha) + B(\alpha) = \alpha \log n + c \alpha^{-3/4}$

根据 Z3，对其求一阶和二阶导数：
$P'(\alpha) = \log n - \frac{3}{4}c \alpha^{-7/4}$
$P''(\alpha) = 0 + \frac{21}{16}c \alpha^{-11/4}$

**凸性与单峰判定**：
对于任意 $\alpha > 0$（由于 $\alpha \in [\alpha_{\text{min}}, \alpha_{\text{max}}]$ 且 $\alpha_{\text{min}}>0$），显然有 $P''(\alpha) > 0$。
引用 **Z1**：因为 $P:(0, \infty) \to \mathbb{R}$ 二阶可导且 $P''(\alpha) > 0$，故 $P(\alpha)$ 在 $(0, \infty)$ 上严格凸。
引用 **Z2**：$P'(\alpha)$ 严格单调递增，必然至多只有一个零点（变号一次），因此 $P(\alpha)$ 是严格单谷函数（即最小化语境下的单峰）。

**解析最优点求解**：
令 $P'(\alpha) = 0$，得到：
$\alpha^* = \left( \frac{3c}{4 \log n} \right)^{4/7}$
*(注：如果 $\alpha^*$ 落在可行区间 $[\alpha_{\text{min}}, \alpha_{\text{max}}]$ 外，由于凸性，区间内的最优解即为离 $\alpha^*$ 最近的端点。)*

**回应 D3（比对 $\alpha_{\text{geo}}$）**：
几何中点为 $\alpha_{\text{geo}} = (\alpha_{\text{min}} \alpha_{\text{max}})^{1/2}$，它的构成要素依赖于 $\epsilon_{\text{bias}}, E_{\text{clip}}$ 以及由多项式率反解决定的 $\alpha_{\text{min}}$。
而 $\alpha^*$ 的量级正比于 $c^{4/7} \propto K_{\text{budget}}^{-1/7}$。二者的标度律完全不同，**差距可以任意大，且无法通过解析化简对齐**。默认取 $\alpha_{\text{geo}}$ 没有任何理论依据。

> **L1 单峰？**【是，严格凸导致的单谷】；
> **$\alpha^*$** = $\left( \frac{3c}{4 \log n} \right)^{4/7}$ ；
> **$\alpha^*$ 与 $\alpha_{\text{geo}}$ 差距** = 无解析必然联系，视具体预算 $K$ 与 $n$ 而定，可能极大偏离。

### 2. L2 处理（泛化层：真实泛化 $G(\alpha)$ 的单峰性分析）

**桥假设 H 定义**：
> ⚠ **工作假设 H**：存在严格单调递减函数 $\psi$，使得 student 模型在真实分布上的泛化性能 $G(\alpha) = \psi(P(\alpha))$。即在预算固定时，Minimax 游戏的可达精度越小，真实分布的泛化表现越好。

**判断可证伪性 / 合理性**：
- **可证伪？** 极易证伪。如 D2 所述，UED 在非凸设定下存在“过度拟合生成课程”的现象。将 $\Phi_\alpha$ 优化至零残差（此时 $P(\alpha)$ 极小）可能导致学生网络彻底遗忘其他未采样任务，反而使 $G(\alpha)$ 崩溃。
- **合理性**：作为严格数学定理，它**不合理且不成立**。作为工程指导，它仅在“欠拟合”阶段（即 $\epsilon_{\text{opt}}$ 占主导时，优化得好能显著提升性能）合理；一旦进入 $\alpha$ 较小的区域（过度专注于困难样本），它必然失效。

**可省实验的最弱可证结论**：
既然 $G(\alpha)$ 的全局单峰性不可证，我们只能退而求其次：
在接受“适度优化残差可以作为隐式正则化”的前提下，$G(\alpha)$ 的真实最优点必定分布在 L1 优化最优解 $\alpha^*$ 附近（因偏离过远会导致要么严重欠拟合，要么引入极大的 Minimax 偏差）。因此，可证结论弱化为：**“寻找泛化极值的搜索中心不应是 $\alpha_{\text{geo}}$，而应是以 $\alpha^*$ 为锚点的局部邻域。”**

> **$G(\alpha)$ 单峰可证？**【否，仅在假设 H 下】；
> **H 是否合理** = 作为定理不成立且可轻易证伪（非凸设定下的课程过拟合），仅能作工程上的工作假设；
> **可省实验的最弱可证结论** = 无需在整个 $[\alpha_{\text{min}}, \alpha_{\text{max}}]$ 内大范围扫点，泛化最优极值被约束在 L1 解析最优解 $\alpha^*$ 的邻域内（作为先验搜索锚点）。

---

### 3. 结尾结论

> **L1 优化层单峰（是），$\alpha^*=\left( \frac{3c}{4 \log n} \right)^{4/7}$；L2 泛化层（仅工作假设且易证伪）；$\alpha$ 轴能否从 3 点降到 1 点（能但需测 2 点验，即不能盲目信赖解析解的绝对泛化性，需在其邻域验证），那起步的 1 个锚点必取 = $\alpha^*$ 而非 $\alpha_{\text{geo}}$。**
