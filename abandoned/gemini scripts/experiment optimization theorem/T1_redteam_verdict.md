## L1：优化层单峰（已证，🟢）

```
P(α)  = α·log n + c·α^{-3/4}              （c = O(K_budget^{-1/4})，由 B4 反解）
P'(α) = log n − (3c/4)·α^{-7/4}
P''(α)= (21c/16)·α^{-11/4} > 0  ∀α>0      ⇒ 严格凸 ⇒ 唯一最小 ⇒ 单峰
α*    = (3c / 4·log n)^{4/7}              ∝ K_budget^{-1/7}
```
顶住了两个鲁棒性追问（D4）：
- **c 弱依赖 α**（经 `Δ_Φ∼α log n`，幂次或非 -3/4）：任意正幂次 `α^{-p}` 仍凸（CHECK 2），单峰不破。
- **换 instance-dependent 紧界**：透视函数保凸（CHECK 4），单峰不破。

## L2：泛化层（证不出，诚实降级）

- **桥假设 H**：`G(α)=ψ(P(α))`，ψ 单调减（可达精度越小泛化越好）。
- **H 易证伪**：UED 非凸设定下，把 `Φ_α` 优化到零残差会**过拟合生成课程**，反使 G(α) 崩。H 只在"欠拟合阶段"（`ε_opt` 主导）合理。**作为定理不成立。**
- **最弱可证结论**：泛化最优**被约束在 `α*` 邻域**（偏太远要么严重欠拟合、要么 Minimax 偏差爆炸）⇒ 搜索中心应是 `α*`，不是 `α_geo`。

## 落到 Stage 3 的实操结论

1. **锚点取 `α* = (3c/4·log n)^{4/7}`**（不取几何中点）。`c` 由 `K_budget` 经 B4 反解、`n` 已知 ⇒ 跑前可直接算（见 `alpha_range_solver.py` 新增 `alpha_star()`）。
2. **α 轴测 2 点**：`α*` + 一个邻域点（建议取 `α*` 的偏欠拟合侧，如 0.5×α* 或 2×α* 中靠近 α_max 的一侧），验证泛化最优确在邻域。
3. **若 `α*` 落在 `[α_min,α_max]` 外**：由凸性，区间内最优 = 离 α* 最近的端点（Gemini 已注明）。
4. **与 T2 衔接**：T2 已证 `α*`（经 λ-无关常数）是 λ-无关锚点 ⇒ T1+T2 合起来"1 锚点跑遍所有 λ"成立。
