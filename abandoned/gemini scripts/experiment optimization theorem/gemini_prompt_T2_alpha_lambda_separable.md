## 0. 记号与对象（Setup）

沿用熵正则异质 score UED 框架（NCC arXiv:2505.20659 Eq.6 符号）。

- `x ∈ X ⊆ ℝ^{d_x}`：student 策略参数（minimax 的 min 变量）。
- `Λ = {l_1,…,l_n}`：level 候选集，`|Λ|=n` 固定有限。
- `y ∈ Δ_n`：adversary 在 n 个 level 上的分布（max 变量）。
- **异质 score**：有 N 类 teacher 各给一个 score 向量 `s^(j)(π_x,Λ) ∈ ℝ^n`，`j=1..N`（如 PVL/learnability/CENIE）。
- **机制混合旋钮 λ**：决定如何把 N 类 score 混成单一 score。混合权重记 `w(λ) ∈ Δ_N`，**混合 score**
  ```
  s̄(x; λ) = Σ_{j=1}^N w_j(λ) · s^(j)(π_x, Λ) ∈ ℝ^n
  ```
  （λ 是 auction 机制参数；w(λ) 是它诱导的 teacher 权重。本任务把 λ 的作用**完全封装在 s̄ 通过 w(λ) 的依赖里**——即 λ 只改变"把哪些 teacher 的 score 怎么线性组合成 s̄"。）
- `α > 0`：熵正则系数（优化稳定性旋钮，温度）。
- `H(y) = -Σ_i y_i log y_i`：Shannon 熵。

**正则化博弈目标**：
```
f_α(x,y; λ) = yᵀ s̄(x; λ) + α H(y)
Φ_α(x; λ) := max_{y∈Δ_n} f_α(x,y; λ)
Φ_0(x; λ) := max_{y∈Δ_n} yᵀ s̄(x; λ)        （未正则化）
```

## 1. 已证地基（你可直接引用，不必重证；但须核对我陈述是否被你用对）

下列均已由 SymPy 机械核验 + 多轮交叉验证坐实（编号 G* / CHECK* 为原始出处标签）：

- **B1（内层解析解，CHECK 2）**：`y*(x;λ) = softmax(s̄(x;λ)/α)`，即 `y*_i ∝ exp(s̄_i/α)`。
- **B2（包络/偏差界，定理2 + Nesterov 2005，CHECK 3）**：
  ```
  Φ_α(x;λ) = α·log Σ_i exp(s̄_i(x;λ)/α)         （LogSumExp）
  0 ≤ Φ_α(x;λ) − Φ_0(x;λ) ≤ α·log n            （∀x, ∀λ）
  ```
  紧界在全等 score 时取到。
- **B3（α 可行区间的两个闭式端点，V4 多项式率）**：给定偏差容忍 `ε_bias`、优化精度 `ε`、预算 `K_budget`、单纯形维数 `n`、**截断底板 `E_clip`**：
  ```
  α_max(λ) = (ε_bias − E_clip(λ)) / log n            （偏差侧：保证 Φ_α−Φ_0 ≤ ε_bias）
  α_min     = (1 / (K_budget · ε⁴))^{1/3}            （预算侧：大 batch 多项式率 K=O(1/(α³ε⁴)) 反解）
  ```
  **关键已证事实**：
  - V4 收敛率 `K = O(1/(α³ε⁴))`（大 batch）/ `O(G_s²σ²Δ_Φ/(α³ε⁵))`（单样本）。**无 e^{2B_s/α}、无 1/ν、无截断**（§7.9 已坐实，对偶 LogSumExp 全局 1-光滑）。
  - 因此 **`α_min` 的闭式不含 `B_s`、不含任何 score 的具体取值**——只含 `K_budget, ε`（单样本版另含 `G_s²σ²Δ_Φ`，仍与 λ 无关，见 §3 待你核）。
- **B4（截断底板的语义，G5）**：`E_clip(λ)` 是 coverage（CENIE）teacher 因把 `s_i=max(-log p_i, -C)` 截断引入的、**与 α 无关**的偏差底板。
  - **PVL / learnability teacher：`E_clip = 0`**（score 本身有界，无需截断）。
  - 含 CENIE 时 `E_clip = E_clip(C) > 0`，且**仅依赖截断阈值 C**（与 α 无关）。

## 2. 为什么要证这个（动机，不要证这段，只为让你理解 claim 的边界）

实验上要在 (α, λ) 二维网格上调参。若能证 **α 的可行区间 `[α_min, α_max]` 不随 λ 移动**（或移动量有界且可控），则可"先定 α、再扫 λ、不回头复核 α-λ 耦合"，省掉坐标式扫描的复核轮。

注意 claim 的**精确边界**：我们**不**要证"泛化最优 α* 不随 λ 漂移"（那是 nonconvex 数据依赖量，证不出）。我们只要证**可行区间端点**（`α_min`、`α_max` 这两个解析量）对 λ 的依赖性——这是纯解析/代数对象，弱得多、可证得多。

## 3. 要证的命题（T2，分 A/B 两档，逐档独立证）

### T2-A（Stage 3a，无 coverage：PVL + SFL learnability）—— 目标：证到"严格可分"🟢

**设定**：所有 teacher ∈ {PVL, learnability}，故 `E_clip(λ) ≡ 0`（B4）。

**要证 T2-A**：在此设定下，α 可行区间 `[α_min, α_max]` 的**两个端点都不依赖 λ**，即
```
∂α_min/∂λ = 0   且   ∂α_max/∂λ = 0,
```
从而区间作为集合 `[α_min, α_max]` 完全不随 λ 移动 ⇒ **α 与 λ 在可行区间层面严格可分**。

**证明应覆盖**：
1. `α_max(λ) = (ε_bias − E_clip(λ))/log n`，代入 `E_clip ≡ 0` ⇒ `α_max = ε_bias/log n`，**显式不含 λ**。但你必须**回到 B2/B3 的推导**确认：`α_max` 的定义里除了 `E_clip` 之外，是否还有任何量经 `s̄(x;λ)` 间接依赖 λ？特别核：偏差界 `Φ_α−Φ_0 ≤ α log n` 的上界常数 `log n` 是否真与 λ 无关（B2 说 ∀λ 成立且紧界只在全等 score 取到——确认 `log n` 这个常数本身不含 s̄）。
2. `α_min = (1/(K_budget·ε⁴))^{1/3}`，**显式不含 λ**。但你必须确认 V4 复杂度 `K=O(1/(α³ε⁴))` 的**前因子 O(·) 里隐藏的常数**是否依赖 λ（即依赖 s̄ 的某个常数如 `G_s, σ², Δ_Φ, B_s`）：
   - 大 batch 版表面上 `K=1/(α³ε⁴)` 不含这些；但 O(·) 里的常数来自 `G_s²σ²Δ_Φ`（单样本版显式、大 batch 版被吸进 O）。**你必须判定**：`G_s`（s̄ 对 x 的 Lipschitz 常数）、`Δ_Φ`（初始次优性）、`σ²`（梯度方差）这三个量是否随 λ 变化？
     - 若它们随 λ 变（因 `s̄=Σw_j(λ)s^(j)` 的 Lipschitz 常数显然依赖 w(λ)），则 `α_min` 严格说**经 O(·) 常数弱依赖 λ**。**这是 T2-A 真正的卡点**，不要糊过去：要么证它们对 λ 的依赖被某个 λ-无关的上界 dominate（如 `G_s(λ)=‖Σw_j∇s^(j)‖ ≤ Σw_j‖∇s^(j)‖ ≤ max_j G_s^(j)`，凸组合被各分量上界控制），从而可用 **λ-无关的最坏情形常数**重定义 `α_min`，使区间端点 λ-无关；要么诚实说"`α_min` 经 O 常数弱依赖 λ，可分只在去掉这些常数的主阶意义下成立"。
3. 结论：在主阶（或在用 λ-无关最坏常数重定义端点后），`[α_min, α_max]` 与 λ 解耦 ⇒ 严格可分。

### T2-B（含 coverage / CENIE）—— 目标：证到"弱耦合有界"🟡

**设定**：含 CENIE，故 `E_clip(λ) = E_clip(C(λ)) > 0`。

**要证 T2-B**：`α_max` 经 `E_clip` 弱依赖 λ，给出
```
∂α_max/∂λ = -(1/log n) · ∂E_clip/∂λ
```
的**显式上界** `|∂α_max/∂λ| ≤ B_couple`，其中 `B_couple` 用 `∂E_clip/∂C · ∂C/∂λ` 的可控量表出。并论证：只要 λ 的调参范围 `Δλ` 满足 `B_couple · Δλ ≤ (某个 α 容差)`，则"先定 α、再扫 λ"导致的 α 区间漂移可被一次性预留的 α 余量吸收，**不需坐标式复核**。
- 这里你需要 `E_clip(C)` 对 C 的单调性/Lipschitz 性。`E_clip(C)` = 截断 `max(-log p, -C)` 相对未截断的偏差底板。直觉：C 越大截断越少、`E_clip` 越小，`∂E_clip/∂C ≤ 0`。给出 `|∂E_clip/∂C|` 的上界（可用分布尾部假设，若需要新假设请显式标注）。
- `α_min` 部分沿用 T2-A 的判定（CENIE 的 `B_s=C` 有界、Lipschitz 常数有界 ⇒ 同样可用 λ-无关最坏常数）。

## 4. 借用零件（可直接引用，不必重证）

- **Z1（包络定理 / Danskin）**：`Φ_α(x;λ)=max_y f_α` 在内层强凹（由 `αH(y)`）下处处可微，`∇_x Φ_α = ∇_x f_α(x, y*(x;λ); λ)`。（用于核 §3 中各量对 x 的依赖，若你需要。）
- **Z2（B2 = Nesterov 2005 偏差界）**：`0 ≤ Φ_α−Φ_0 ≤ α log n`，instance-dependent 紧界 `α log(1+(n-1)e^{-Δ/α})`，Δ=次优 score 间隙。
- **Z3（凸组合的 Lipschitz 常数次可加）**：若每个 `s^(j)` 对 x 是 `G_s^(j)`-Lipschitz，则 `s̄=Σw_j s^(j)`（`w∈Δ_N`）满足 `G_s(λ)=Lip(s̄) ≤ Σw_j G_s^(j) ≤ max_j G_s^(j)`。（这是你在 §3-A.2 把 λ-依赖常数 dominate 成 λ-无关上界的主要工具——但请显式验证它对你用到的具体量都成立。）

## 5. 禁糊清单（任何一步用到下列之一，必须显式展开或显式引用编号结论，不许 "standard"/"clearly"/"类似地"/"取决于" 带过）

- **(a)** 任何"X 不依赖 λ"的断言 —— 必须回到 X 的**定义式/闭式**，逐个变量检查 λ 是否经某中间量（尤其经 `s̄=Σw_j(λ)s^(j)`）潜入。**最常见的糊点**：声称 `α_min=(1/(K_budget ε⁴))^{1/3}` 不含 λ，却忽略 O(·) 前因子里的 `G_s(λ),σ²(λ),Δ_Φ(λ)`。这一项必须正面处理（§3-A.2）。
- **(b)** 把 O(·) 里的常数当成"与一切无关" —— 禁止。V4 大 batch 率 `K=O(1/(α³ε⁴))` 的 O 里藏着 `G_s²σ²Δ_Φ`，你必须判定它们的 λ-依赖，并明确是"被 λ-无关上界 dominate（可分）"还是"残留 λ 依赖（弱耦合）"。
- **(c)** 偏导/单调性断言（如 `∂E_clip/∂C ≤ 0`）—— 必须给推导或显式假设；用到分布尾部性质时把假设写出来。
- **(d)** "区间不移动 ⇒ 不需复核" 这步逻辑 —— 必须说清：是端点严格不动（T2-A），还是端点漂移被 α 余量吸收（T2-B）。两者结论强度不同，不许混。
- **(e)** 若某处你只能证到"主阶可分、O 常数弱依赖"，**如实降级**，不要 claim 成"严格可分"。
