# 证明任务书 T3：multi-winner ⊇ single-winner 单调 ⇒ w 轴只测两端点

> **致证明者（Gemini 3.1 Pro）**：这是一个**自包含**的证明任务，不依赖你之前任何对话。下面给全了记号、已证地基、要证命题、借用零件、禁糊清单。**你的全部任务是评估并（在可证的层面上）证明 §3 的 T3，不要重写其他定理、不要重新发明地基。** 遵守 §6 禁糊清单。
>
> ⚠ **首要前提：T3 的"测一个 fractional 点就够"很可能只在一个非平凡条件下成立，而那个条件可能不成立。** 本命题有三个强度层次（见 §3），**绝大多数"看似证明 T3"的论证其实只证到了平凡层 L0，对省实验毫无用处**：
> - **L0（平凡，不算数）**：`sup_{w∈Δ_N} G(w) ≥ G(one-hot)`。这是"集合上的 max ≥ 集合里某一点"，**恒真且无信息**——它只说"存在某个 fractional 不差于 argmax"，**不保证你实际测的那个 fractional 点 ≥ argmax**。**严禁把 L0 当成 T3 的证明。**
> - **L1（连续性）**：`G(w)` 在单纯形 `Δ_N` 上连续。保证 max 可达、L0 有意义。🟢 大概率可证。
> - **L2（真命门）**：**one-hot 顶点（argmax）是否"非局部最优"**——即从顶点沿单纯形内部方向走，目标是否**严格上升**（`∂G/∂w` 在顶点指向内部）。**只有 L2 成立，"测一个内部 fractional 点 > argmax"才有保证，w 轴才真能省。** 这需要算方向导数，依赖 score 之间的相关结构，🔴 可能算不出或可能为假。
>
> **你的任务不是停在 L0，而是：(1) 证 L1（连续性）；(2) 正面攻 L2——在一个可解析的代理目标上算 one-hot 顶点的方向导数，判定顶点是否非局部最优，并诚实标出"代理→真实泛化"的桥假设。** 给出"L2 在条件 Z 下成立、否则顶点可能就是最优（multi-winner 无增益）"远胜于一个停在 L0 的空证明。**先证伪、证伪不成再证明**：见 §2.5。

## 0. 记号与对象（Setup）

沿用熵正则异质 score UED 框架（NCC arXiv:2505.20659 Eq.6 符号）。

- `x ∈ X ⊆ ℝ^{d_x}`：student 策略参数。`Λ={l_1,…,l_n}`：level 候选集，`|Λ|=n`。
- `y ∈ Δ_n`：adversary 在 n 个 level 上的分布。`α>0`：熵正则系数（温度）。
- **N 类 teacher**，各给 score 向量 `s^(j)(π_x,Λ) ∈ ℝ^n`，`j=1..N`（如 PVL/SFL/CENIE）。各 `‖s^(j)‖_∞ ≤ B_s`。
- **multi-winner 权重 `w ∈ Δ_N`**（N 维概率单纯形，**注意：这是 teacher 之间的权重，维度 N，不是 level 上的 y∈Δ_n**）。混合 score
  ```
  s̄(x; w) = Σ_{j=1}^N w_j · s^(j)(π_x, Λ) ∈ ℝ^n
  ```
- **single-winner / argmax = one-hot 顶点**：`w = e_k`（某个 teacher 独占），此时 `s̄ = s^(k)`。
- **fractional / multi-winner**：`w` 落在 `Δ_N` 内部（多个 teacher 加权）。
- **关系链（main.tex §4.3.4 ⚑V2，已立）**：single-winner ⊂ top-k ⊂ fractional，argmax = `w=one-hot` 退化特例。

**内层解析解（已证）**：`y*(x; w) = softmax( s̄(x;w) / α ) = softmax( (1/α) Σ_j w_j s^(j)(x) )`。w 连续进入。

**目标量**：`G(w)` = student 在混合 score `s̄(·;w)` 下训练收敛后，在真实任务分布上的泛化性能（越大越好）。

## 1. 已证地基（可直接引用，不必重证；但须核对我陈述是否被你用对）

- **B1（内层解析解）**：`y*(x;w)=softmax(s̄(x;w)/α)`，`s̄(x;w)=Σ_j w_j s^(j)(x)`。w 经 s̄ 线性进入 logits。
- **B2（课程分布由混合 score 的熵/方差决定）**：采样课程 `y*` 的有效支撑（多样性）可由 `H(y*)` 或 `y*` 的分散度刻画。`s̄` 越分散（方差大）⇒ softmax 后 `y*` 越尖锐/越有结构；`s̄` 越平 ⇒ `y*` 越均匀。**这是把"w 怎么混 score"接到"课程长什么样"的桥（定性，T3 要把它对 w 求导定量化）。**
- **B3（紧凸性）**：`Δ_N` 是紧凸集（N-单纯形），其顶点恰为 `{e_1,…,e_N}`（one-hot），内部为严格凸组合。
- **B4（与 T1/T2 衔接）**：α 已由 T1 定在 `α*`（λ-无关锚点）。本任务在固定 α=α* 下讨论 w，不重新扫 α。

## 2. 为什么要证这个（动机，不要证这段）

§4.3.4 已立关系链 single ⊂ top-k ⊂ fractional。实验上 argmax 那点 **Stage 2 已跑、可复用**。若能证"沿 one-hot 顶点往内部走目标上升"（L2），则 w 轴只需测 argmax（已有）+ **一个**内部 fractional 点，中间 top-k 免测（w 轴 ×3→×2，且 argmax 点免费）。**关键**：省实验靠的不是 L0（"存在某 fractional 更好"），而是 L2（"往内部走一定更好" ⇒ 测的那个内部点真的 > argmax）。

## 2.5 先证伪（在动笔证明前必做）

## 3. 要证的命题（T3，分 L1/L2 两层 + 显式拒绝 L0）

### L0（平凡，**不要把它当结论**）
`sup_{w∈Δ_N} G(w) ≥ G(e_k)`——恒真，无信息。**任务书要求你显式写一句"L0 平凡、不构成 T3 的省实验依据"，然后跳过它。**

### L1（连续性）—— 目标：证出 🟢
**要证 L1**：`G(w)` 在 `Δ_N` 上连续，从而 `max_{Δ_N} G` 可达。
- 路径：若链条 `w ↦ s̄(x;w) ↦ y*(x;w)=softmax(s̄/α) ↦ 训练动力学 ↦ G` 每一环连续，则 G 连续。`w↦s̄` 线性（连续）；`s̄↦softmax` 光滑（连续）。**真正要核的是"训练收敛点对课程分布 y* 的连续依赖"**——这一环在 nonconvex 训练下可能只有"几乎处处连续"或需要额外假设。如实标注你用了什么（如：收敛点对 y* 的 Lipschitz/连续依赖作为假设）。

### L2（顶点非局部最优）—— 目标：在可解析的**代理目标**上正面攻，🔴 主攻点
直接对真实泛化 `G(w)` 算方向导数大概率不可行（nonconvex、数据依赖）。**改在代理目标上算**，再用桥假设接回 G。

**定义代理目标**（课程多样性，可解析）：取
```
D(w) := Var_i( s̄_i(x;w) )   或   H(y*(x;w))   （二选一或都做；Var 更易解析）
```
（直觉：`s̄` 的方差越大 ⇒ softmax 课程越有结构/越能区分难易 ⇒ 多样性代理越高。）

**要证 L2**：计算 one-hot 顶点 `e_k` 处、沿内向方向 `v = e_j - e_k`（混入 teacher j）的**单边方向导数** `D_v D(e_k)`，给出其符号的**充要/充分条件**，该条件用 **score 之间的相关/协方差** 表出。
- 以 `D(w)=Var_i(s̄_i)` 为例展开：`s̄ = Σ_j w_j s^(j)`，`Var_i(s̄)=wᵀ Σ w`，其中 `Σ_{jk}=Cov_i(s^(j), s^(k))`（teacher 间在 level 维度上的 score 协方差矩阵，N×N）。这是 w 的**二次型**——可解析求方向导数。
- 在顶点 `e_k`：`D(e_k)=Σ_{kk}=Var_i(s^(k))`。沿 `v=e_j-e_k` 的方向导数 `D_v D(e_k) = ?`（请显式算这个二次型的方向导数）。
- **给出"顶点非局部最优"（`D_v D(e_k) > 0`，即混入 j 增大多样性代理）的条件**，用 `Σ_{jj}, Σ_{kk}, Σ_{jk}`（或相关系数 `ρ_{jk}`）表出。直觉预期：当 teacher j 与 k **不高度正相关**（`ρ_{jk}` 小）时混入 j 增大方差 ⇒ 顶点非局部最优 ⇒ multi-winner 有增益；当 `ρ_{jk}→1`（j 被 k 包含）时无增益（这正好呼应 T4 的冗余判据）。
- **桥假设**：写出"代理目标 `D(w)` 改善 ⇒ 真实泛化 `G(w)` 改善"所需的最小假设 H_T3，判其可证伪性（呼应 D2），明确它是定理还是工作假设。

## 4. 借用零件（可直接引用）

- **Z1（紧集上连续函数取到 max）**：`Δ_N` 紧 + `G` 连续（L1）⇒ `max_{Δ_N} G` 可达。这是 L0/L1 的全部内容，**不要在此之外引申**。
- **Z2（二次型的方向导数）**：`D(w)=wᵀΣw`，则 `∇D(w)=2Σw`，沿单位方向 `v` 的方向导数 `D_v D(w)=⟨∇D(w), v⟩=2 wᵀΣv`。在顶点 `e_k`：`D_v D(e_k)=2 e_kᵀ Σ v = 2 (Σ_{kj} - Σ_{kk})`（当 `v=e_j-e_k`）。**请独立验证这个展开，不要直接抄。**
- **Z3（协方差矩阵半正定）**：`Σ⪰0`，`Σ_{jk}=ρ_{jk}√(Σ_{jj}Σ_{kk})`，`ρ_{jk}∈[-1,1]`。
- **Z4（与 T4 的衔接）**：本任务算出的 score 协方差 `Σ`，正是 T4（异质增益 ∝ score 相关）要用的同一个对象——若此处证出"`ρ_{jk}` 大 ⇒ 顶点局部最优 ⇒ 无增益"，则与 T4 的冗余判据互证。
