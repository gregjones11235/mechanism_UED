# 证明任务书 T1：α 泛化曲线单峰 ⇒ α 轴从扫 3 点降为解析定 1 点

> **致证明者（Gemini 3.1 Pro）**：这是一个**自包含**的证明任务，不依赖你之前任何对话。下面给全了记号、已证地基、要证命题、借用零件、禁糊清单。**你的全部任务是评估并（在可证的层面上）证明 §3 的 T1，不要重写其他定理、不要重新发明地基。** 遵守 §6 禁糊清单。
>
> ⚠ **首要前提：T1 的"泛化单峰"很可能根本证不出。** 这是出题方的**预判**，不是要你迎合——而是要你**精确地把"能证的层"和"证不出的层"切开**。本命题分两层（见 §3）：
> - **L1（优化层）**：α 的"可达精度"`P(α)`（算法在预算内能把正则化目标 `Φ_α` 优化到多好 + 它离真目标 `Φ_0` 多远，二者之和）是否单峰。**这一层很可能可证**——它是两个符号相反的解析单调量之差。
> - **L2（泛化层）**：student 在**真实任务分布**上的泛化 `G(α)` 是否单峰。**这一层大概率证不出**——`G(α)` 是 nonconvex、数据依赖的经验量，"优化得好 + 偏差小 ⇒ 泛化好"在 nonconvex 下不是定理。
>
> **你的任务不是硬证 L2，而是：(1) 干净地证 L1（若它真成立）；(2) 精确陈述"从 L1 推到 L2 需要的最小额外假设"，并判断该假设是否可证伪/是否合理，而不是默认它成立。** 给出"L2 在假设 H 下成立、H 本身证不出只能作工作假设"远胜于给出一个把 H 偷偷当真的"已证 G(α) 单峰"。**先证伪、证伪不成再证明**：见 §2.5。

## 0. 记号与对象（Setup）

沿用熵正则异质 score UED 框架（NCC arXiv:2505.20659 Eq.6 符号）。

- `x ∈ X ⊆ ℝ^{d_x}`：student 策略参数。`Λ={l_1,…,l_n}`：level 候选集，`|Λ|=n`。
- `y ∈ Δ_n`：adversary 分布。`s(π_x,Λ)∈ℝ^n`：score 向量，`‖s‖_∞ ≤ B_s`。
- `α>0`：熵正则系数（温度）。`H(y)=-Σ y_i log y_i`：Shannon 熵。
- `f_α(x,y)=yᵀs+αH(y)`，`Φ_α(x)=max_{y∈Δ_n} f_α(x,y)`，`Φ_0(x)=max_y yᵀs`。
- **可行区间**：`α ∈ [α_min, α_max]`，两端点闭式见 B3。**本任务在此固定区间内讨论 α 的单峰性。**
- **几何中点**：`α_geo := √(α_min · α_max)`。

## 1. 已证地基（可直接引用，不必重证；但须核对我陈述是否被你用对）

- **B1（内层解析解）**：`y*(x)=softmax(s(x)/α)`。
- **B2（偏差界，Nesterov 2005）**：`0 ≤ Φ_α(x)−Φ_0(x) ≤ α log n`，∀x。**关于 α 单调增**（线性，斜率 `log n>0`）。instance-dependent 紧界 `α log(1+(n-1)e^{-Δ/α})`，Δ=次优 score 间隙。
- **B3（α 可行区间端点）**：`α_max=(ε_bias−E_clip)/log n`；`α_min=(1/(K_budget·ε⁴))^{1/3}`（大 batch 多项式率反解）。
- **B4（V4 收敛复杂度，无截断、无指数）**：
  ```
  K(α, ε) = O( 1/(α³ ε⁴) )                  （大 batch）
  K(α, ε) = O( G_s²σ²Δ_Φ/(α³ ε⁵) )          （单样本）
  ```
  **反过来读**：在固定预算 `K_budget` 步内，算法能把 `Φ_α` 优化到的精度 `ε_opt(α)` 满足 `K_budget = O(1/(α³ ε_opt⁴))` ⇒ `ε_opt(α) = O( (1/(α³ K_budget))^{1/4} ) = O( α^{-3/4} K_budget^{-1/4} )`。即**预算内的优化残差关于 α 单调减**（α 越大越好优化）。

## 2. 为什么要证这个（动机，不要证这段）

实验上现在的方案是"在 `[α_min,α_max]` 内扫 3 个 α 锚点做 active-search 找峰"。若能证 α 的目标曲线在区间内**单峰（拟凹）**，则最优 α* 附近无需精扫——固定在某个解析可定的点（如几何中点 `α_geo`）即够，省掉 2 个 α 取值 × 全部下游配置（α 轴 ×3→×1，墙钟 26 天→8.7 天）。

**关键**：省实验只需要"目标曲线单峰 + 峰位置可解析估"。**单峰**保证"不会错过更好的 α"；**峰位可估**保证"那 1 个点就该取在哪"。

## 2.5 先证伪（在动笔证明前必做）

下面 D1/D2/D3 是出题方预想的"T1 会塌"的路径，**但绝不穷尽——不要被框住**。先逐条裁决，**再用你自己的视角独立搜一遍 D1–D3 之外的证伪路径**。

- **D1（L1 的两个单调分量之差不单峰）**：可达精度 `P(α)=（偏差，关于α增）+（优化残差，关于α减）`。两个单调反向的量之差**不一定单峰**（可能有多个驻点 / 平台 / 多峰）。检查：`P'(α)` 在 `[α_min,α_max]` 内**是否至多变号一次**（单峰的等价定义）？把 `P'(α)` 写出来，看它能否被证明单调，或至多一个零点。**若 P'(α) 可在区间内多次变号，L1 就不是单峰，T1 在优化层即塌。**
- **D2（L1→L2 的桥假设其实是假的）**：从"可达精度单峰"推"泛化单峰"，要求"泛化随可达精度单调"这个桥。但 UED 里**采样到错误的课程**可能让"偏差更小/优化更好"反而泛化更差（过拟合到 Φ_α 的特定驻点）。检查：这个桥假设有没有一个**具体的反例情景**（哪怕是 toy）能让它失效？若有，L2 不可证，只能作工作假设——明确说出来。
- **D3（峰位置不可解析估，即便单峰）**：即便 `P(α)` 单峰，其峰位 `α*=argmin P` 可能落在**远离几何中点 `α_geo`** 的地方，使"取 `α_geo` 一个点"漏掉真峰。检查：`α* = argmin P(α)` 的解析位置是什么？它与 `α_geo=√(α_min α_max)` 的关系如何？若 `α*` 与 `α_geo` 差很远，则"单峰"成立但"取中点省实验"仍不成立——必须给出**正确的解析峰位**或一个含峰的小区间。
- **D4（你自己找的，强制项）**：在 D1–D3 之外主动找反例/隐藏前提（可能方向，但不限于此：B2/B4 我转述时用错了条件；`ε_opt(α)` 的反解在 `α→α_min` 边界退化；instance-dependent 紧界 `α log(1+(n-1)e^{-Δ/α})` 相对 `α log n` 改变了偏差项的凹凸性从而改变 P 的单峰性；`Δ_Φ` 本身依赖 α 使 §1-B4 的反解不是纯 `α^{-3/4}`）。**即使没找到，也必须写一句"已独立搜索 D1–D4 之外的证伪路径，结果为 ___"，不许跳过。**

> 若任一条塌，明确说是哪一层（L1/L2）、什么条件、退路是什么，然后只对没塌的层继续 §3。

## 3. 要证的命题（T1，分 L1/L2 两层，逐层独立处理）

### L1（优化层："可达精度"单峰）—— 目标：干净证出 🟡→🟢

定义**可达精度泛函**（在固定预算 `K_budget`、固定 ε 设定下，作为 α 的函数）：
```
P(α) := [ Φ_α − Φ_0 的偏差上界 ]  +  [ 预算 K_budget 内的优化残差 ε_opt(α) ]
      = α·log n  +  c · α^{-3/4}                  （c = O(K_budget^{-1/4})，由 B4 反解）
```
（直觉：`P(α)` 越小 ⇒ 算法越能既忠于真目标 `Φ_0` 又把它优化好。α 大 → 偏差大但好优化；α 小 → 偏差小但难优化。`P` 是这两害之和。）

**要证 L1**：`P(α)` 在 `[α_min, α_max]`（或在 `(0,∞)`）上**单峰（这里是单谷/拟凸，因 P 是"害"，要最小化）**，即 `P'(α)` 至多变号一次；并给出**解析最优点** `α* = argmin P(α)` 的闭式。
- 提示路径（不强制）：`P'(α) = log n − (3c/4)·α^{-7/4}`。令 `=0` 解 `α*`。验证 `P''(α) > 0`（凸 ⇒ 严格单谷 ⇒ 单峰）：`P''(α)=(21c/16)α^{-11/4} > 0` 对 α>0 恒成立 ⇒ **P 严格凸 ⇒ 全局唯一最小 ⇒ 单峰**。**若如此，L1 是干净的凸分析，请把每步代数写满、确认凸性。**
- **但你必须核 §2.5-D4**：`c` 是否真与 α 无关（`Δ_Φ` 若依赖 α 则 `c=c(α)`，破坏纯幂次，凸性需重验）；`α log n` 换成 instance-dependent 紧界后是否仍凸（D4）。
- 给出 `α*` 后，**回答 D3**：`α*` 与几何中点 `α_geo=√(α_min α_max)` 差多远？"取 `α_geo` 一点"是否就近似取到了 `α*`？若不是，给出该取的正确解析点。

### L2（泛化层："真实泛化" `G(α)` 单峰）—— 目标：界定假设、判可证伪，**不硬证**

`G(α)` = student 用混合 score 在温度 α 下训练收敛后，在**真实任务分布**上的泛化性能。**不要求你证它单峰。** 要求：
1. **精确写出"L1 单峰 ⇒ L2 单峰"所需的最小桥假设 H**（形如"泛化 `G` 是可达精度 `P` 的单调函数 `G=ψ(P)`，ψ 单调"，或更弱的形式）。把 H 写成可检验的数学陈述。
2. **判断 H 是否可证伪 / 是否合理**：H 在 nonconvex、数据依赖的 UED 下成立吗？给出"H 失效"的具体情景（呼应 D2）。明确：H 是**定理**还是只能作**工作假设**？
3. 若 H 只能作工作假设，**给出一个比"G 单峰"更弱、但仍足以省实验的可证结论**（例如："G 在 `α*` 的某邻域内随 P 单调下降"这种局部陈述，或"G(α_geo) 有解析上界"）。

## 4. 借用零件（可直接引用）

- **Z1（凸函数的单谷性）**：若 `P:(0,∞)→ℝ` 二阶可导且 `P''(α)>0`，则 P 严格凸 ⇒ 至多一个驻点且为全局最小 ⇒ 单谷（单峰的最小化版本）。
- **Z2（两单调量之差的驻点）**：`P=A(α)+B(α)`，`A'>0`、`B'<0`，则 `P'=A'+B'` 的零点即 `A'(α*)=-B'(α*)`。零点唯一性由 `A'+B'` 单调（即 `P''` 不变号）保证。
- **Z3（B2/B4，见 §1）**：偏差 `α log n` 关于 α 线性增（凸）；优化残差 `ε_opt(α)∝α^{-3/4}` 关于 α 减且凸（α>0 时 `α^{-3/4}` 凸）。
