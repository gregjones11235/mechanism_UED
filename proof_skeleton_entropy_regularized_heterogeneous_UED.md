## 0. 记号与对象（Setup）

沿用 NCC（arXiv:2505.20659）Eq.6 的符号。

- `x ∈ X ⊆ ℝ^{d_x}`：student 策略参数（minimax 的 min 变量）。`π_x` 为对应策略。
- `Λ = {l_1, …, l_n}`：当前 level 缓冲/候选集，`|Λ| = n`，固定有限。
- `y ∈ Δ_n`：adversary 在 n 个 level 上的分布（minimax 的 max 变量），`Δ_n` 为概率单纯形。
- `J(π_x, Λ) ∈ ℝ^n`：分量 `J_i = ` 策略 `π_x` 在 level `l_i` 上的期望回报（return）。
- `s(π_x, Λ) ∈ ℝ^n`：**score 向量**，分量 `s_i` 为 level `l_i` 的得分。
  - **NCC 的 zero-sum 情形**：`s = -J`（score = 负回报）。
  - **本任务的异质情形**：`s = s_het`，其中 `s_het` 取自 {PVL, Gen-SFL learnability, CENIE coverage} 或其混合。
- `H(y) = -Σ_i y_i log y_i`：单纯形上的 Shannon 熵（强凹）。
- `α > 0`：熵正则系数（NCC 已有的旋钮）。

**正则化博弈目标**（NCC Eq.6 的形式，score 换成异质）：
```
f_α(x, y) = yᵀ s(π_x, Λ) + α H(y)
min_{x∈X} max_{y∈Δ_n} f_α(x, y)
```
定义 `Φ_α(x) := max_{y∈Δ_n} f_α(x, y)`（外层目标）。
**未正则化目标**（你「真正想优化」的异质目标）：
```
Φ_0(x) := max_{y∈Δ_n} yᵀ s(π_x, Λ)
```

> **核心区分（来自 deep-research caveat 2，🔴 承重）**：strong-concavity 来自 `α H(y)`，**不来自** `s` 是不是 `-J`。`s` 仅以**线性项** `yᵀ s` 进入 `f_α`。因此「`f_α` 对 `y` 强凹」对**任意有界 `s`** 都成立——这是整个出路的支点。**这一句必须独立验证：确认 `s` 确实只通过 `yᵀs` 这个对 y 线性的项进入目标；若某 score（如 CENIE，见 §4）实际作用在采样分布层而非此线性项，则本框架对它不适用。**

| **A2（score 有界）** | 存在 `B_s < ∞` 使 `‖s(π_x,Λ)‖_∞ ≤ B_s`，∀x | 偏差 bound 的核心，且保证 `Φ_0,Φ_α` 有限 | PVL: 待核（回报有界则有界）；learnability `p(1-p)∈[0,1/4]`: ✅；CENIE: 🔴 log-密度可能无界，须截断 |

## 2. 主定理（形式化陈述）

### 定理 1（收敛，🟢 套路 + 🟡 一处需坐实）
在 A1–A6 下，对**任意**满足 A2/A3/A5 的 score `s`（含异质），熵正则博弈 `min_x max_y f_α(x,y)` 上的随机 two-timescale GDA 在 `K` 步后满足
```
E[ ‖∇Φ_α(x_K)‖² ] ≤ O(K^{-1/2})    （或 Lin-Jin-Jordan 给出的具体率）
```
即收敛到 `Φ_α` 的 ε-近似一阶驻点（ε-approximate first-order Nash）。

- **🟢 依据**：`f_α(x,·)` 在 `Δ_n` 上 `α`-强凹（由 `αH(y)`，A5 保证 score 项线性），`f_α(·,y)` 关于 `x` 是 nonconvex 但 ℓ-smooth（由 A3）。这正是 Lin-Jin-Jordan 2020（arXiv:1906.00331）Thm 4.5 的 nonconvex-**strongly**-concave 设定。
- **🟡 需坐实**【⚑ 已被 V2 修正，见顶部】：~~须核对 `αH(y)` 在边界是否满足一致强凹常数~~ —— **V2 更正：强凹常数全局不退化（Hessian `⪯ -αI`）；退化的是 smoothness（`y_i→0` 时 Hessian 上界爆炸）。** 修复不是限内部紧子集，而是**改用 two-timescale stochastic mirror descent（内层 KL-mirror ascent），`αH(y)` 天然匹配 KL、不需 y-Lipschitz**。剩余 🟡：该完整组合（两时间尺度+随机+NC-SC+KL）无单篇现成定理，需拼接/局部自证。

### 定理 2（偏差有界，🔴 承重 —— 本任务真正的 novelty 与命根子）
在 A1–A2 下，对任意 `x`，
```
0 ≤ Φ_α(x) − Φ_0(x) ≤ α · log n
```
且正则化解 `x_α^* ∈ argmin Φ_α` 与未正则化解 `x_0^* ∈ argmin Φ_0` 满足
```
| min_x Φ_α(x) − min_x Φ_0(x) | ≤ α · log n  →  0   (α→0)
```

- **🔴 为何是承重**：这一步把「收敛到的点」与「你真正想要的异质目标最优」连起来。**没有它，定理 1 只证明了「会停下来」，但停的地方可能不是你的异质目标的解——整个出路三的价值就落空。**
- **证明骨架（🟡 套路上界 + 🔴 待核下界与紧性）**：

two-timescale GDA 收敛到的 `x_K` 满足：它是 `Φ_α` 的 ε-驻点，且 `Φ_α` 与异质目标 `Φ_0` 的最优值差 `≤ α log n`。**因此 α 是一个可解析控制的「收敛性 ↔ 异质保真度」权衡旋钮**：α 大 → 收敛好但偏离异质目标多；α→0 → 逼近异质目标但定理 1 的强凹常数退化、收敛变慢。**这个 α-权衡的解析刻画本身就是可发表的 constructive 结果**，且取代了「靠满种子实验刻画崩溃边界」（见 [idea可行性分析.md](idea可行性分析.md) §4.5.2 的成本论证）。

### 7.1 目标定理（要证的东西，精确陈述）

设 `f_α(x,y) = yᵀ s(π_x,Λ) + α H(y)`，`Φ_α(x) = max_{y∈Δ_n} f_α(x,y)`。在假设 A1–A6（§1，且 A2/A3/A5 已对所用 score 核验通过）下，考虑如下**随机两时间尺度算法**：
- **内层（y，KL-mirror ascent / exponentiated gradient）**：`y_{t+1} = argmax_{y∈Δ_n} { η_y ⟨ŷ-grad, y⟩ − D_KL(y ‖ y_t) }`，其中 ŷ-grad 为 `∇_y f_α` 的无偏随机估计；
- **外层（x，SGD）**：`x_{t+1} = x_t − η_x ∇̂_x f_α(x_t, y_{t+1})`；
- **步长比**：`η_x / η_y → 0`（外慢内快，两时间尺度）。

**要证**：存在步长选取使得 `(1/K) Σ_{t<K} E‖∇Φ_α(x_t)‖² ≤ ε`，迭代复杂度 `K = O(ε^{-?} · poly(κ, n, σ², B_s, ...))`，其中 `κ` 为**相对条件数**（见 G2）。**只要求一阶驻点（ε-stationary of Φ_α），不要求 global Nash**（V2 已定，§2 推论）。

### 7.2 借用零件（证明者可直接引用，不必重证；但须核对假设匹配）

- **Z1（外层 oracle 的光滑性）**：若 `f_α(x,·)` 对 y 强凹、且 `f_α(·,·)` 对 (x,y) 满足联合 Lipschitz/光滑（A3），则 `Φ_α(x)=max_y f_α` 是 `L_Φ`-smooth，且 `∇Φ_α(x)=∇_x f_α(x, y*(x))`（Danskin / 包络定理）。**这是 TTGDA 分析的标准起点**（Lin-Jin-Jordan 2020, arXiv:2408.11974/1906.00331）。
- **Z2（NC-SC 两时间尺度率）**：欧氏 TTGDA 在 NC-SC + 光滑下达 `O(ε^{-2})`（确定）/相应随机率（Lin-Jin-Jordan；Zhang et al. UAI 2021, arXiv:2103.15888 给下界与加速）。**借其证明骨架，不借其欧氏光滑假设**（正是该假设在边界失效，见 G2）。
- **Z3（相对光滑 / Bregman 一阶下降引理）**：mirror descent 在「目标相对于 Bregman reference `h` 满足 relative smoothness / relative strong convexity」时收敛，**不需要目标自身梯度 Lipschitz**（Lu-Freund-Nesterov relative smoothness；Bregman SMD，arXiv:1803.07374、2104.09813）。**这是绕开边界 Lipschitz 的正确数学语言。**
- **Z4（熵偏差）**：`Φ_α − Φ_0 ≤ α log n`（Nesterov 2005，§2 定理 2，已坐实）。

### 7.3 必须自证的缝隙（G1–G5）——这才是论文的理论 contribution

> 这些是 Z1–Z4 **没有覆盖、必须新补**的连接处。每条都给出「要证什么」+「为什么现成定理不够」+「建议工具」。

- **G1（内层几何切换的合法性）**：把内层从「欧氏投影梯度上升」换成「KL-mirror ascent」后，Z1 的包络性质（`∇Φ_α = ∇_x f_α(x, y*)`、`y*(x)` 的 Lipschitz 依赖于 x）是否仍成立？
  - 要证：`y*(x)` 在 **KL/单纯形几何**下对 x 的稳定性（`‖y*(x)-y*(x')‖` 或 `D_KL` 意义下），它是外层 `L_Φ`-smooth 的前提。
  - 为何不够：Z1 的 `y*` 稳定性是在欧氏强凹下推的；KL 几何下要用 relative strong convexity 重推。
  - 工具：Z3 + 包络定理在 Bregman 几何下的版本。
  - 🟡 **经验探针（理论假设 ↔ 实验：`h_drift_check`）**：G1 / 禁糊 §7.4-(b) 要求的「`y*(x)` 对 x 稳定」是**纸上假设**，须在真实 run 上体检。其经验对应物是 `sigma_measure.py:h_drift_check`——
    - **桥接**：x = student 参数（随训练变化）；`y*(x)` = 给定 x 时内层在单纯形上的最优采样分布，由 N 类 estimator 的 score 协方差 `Σ` 决定。`Σ`/相关矩阵 `ρ` 是 `y*(x)` 结构的**可观测代理**。「`y*(x)` 对 x Lipschitz 稳定」翻译成实验语言 = **`ρ` 沿训练时点（早/中/末多个 checkpoint）不剧烈跳变**。
    - **判读**：`h_drift_check` 在多 checkpoint 测 `ρ`，逐对算极差 < tol → 稳定 → G1/(b) 的稳定性假设在该设定下**经验上未被违反** → 用一次 `Σ` 测量定 Stage3 配置（T3/T4 判据）合法；极差 ≥ tol → `y*(x)` 漂移 → **G1 假设被违反**，定理保证打折，须退回事后消融（真跑对比实验），不可靠一次测量预测全程。

## 8. 定理的适用边界：引入「会训练的生成器 teacher」是否破坏收敛（2026-06-19）

> **触发**：作者问「假如后续引入多个 teacher **agents**（PAIRED 式会训练的生成器网络），收敛定理是否仍成立」，以及「现在这套不也已经是 general-sum 吗」。本节钉死定理的适用边界，区分**两层 general-sum**，并给出加生成器 teacher 的代价与唯一保住定理的路线。结论用 §0 符号（`x`=student 参数，`y∈Δ_n`=adversary 分布，`s`=score）。

### 8.1 现在这套**已经是 general-sum**（non-zero-sum）—— 这正是 novelty，不是缺陷

main.tex §2（第146行）白纸黑字：learnability/coverage 与 regret「统计上不同、不能写成 regret+小扰动」，故混合它们的异质 population「**no longer a zero-sum game**」，正是 NCC 留作 future work 的 **non-zero-sum** 情形。所以作者的判断对：**现在这套在博弈论意义上就是 general-sum。**

→ 与 §279(i)/§0(§101) 一致：novelty = 「给 non-regret 异质 score 配收敛保证」，覆盖的恰是 general-sum 目标。

### 8.2 关键区分：**两层 general-sum**（这是回答全部疑问的支点）

general-sum 出现在**哪一层**，决定定理成立与否：

| | **目标层 general-sum**（现状） | **决策结构层 general-sum**（加生成器 teacher） |
|---|---|---|
| 表现 | score `s` 异质、非 `-J` | score 异质 **+ 对手决策变量变性** |
| general-sum 装在哪 | 全部塞进 **score 向量 `s`** | 塞进 score **+ 对手的优化对象** |
| 对手决策变量 | `y∈Δ_n`（单纯形，固定有限 buffer） | `φ`（生成器网络参数，**非凸**） |
| 强凹性来源 | `αH(y)` 给 `y` 强凹 ✅ | 熵给不了 `φ` 强凹 ❌ |
| **定理是否成立** | ✅ 成立 | ❌ 不成立（在现证明下） |

**机制（§0 第101行 + §279(i) 的内核）**：本工作把 general-sum 的非零和性**全部塞进 score 向量 `s`**，而 `s` **只通过对 `y` 线性的项 `yᵀs` 进入** `f_α`（A5）。强凹性由 `αH(y)` 在**单纯形 `y`** 上独立提供，**与 `s` 是什么完全解耦**。于是：

> **现状 = 「general-sum 的目标，装在 zero-sum 的 min–max 求解形式里」。** non-zero-sum 体现在 `s` 的异质性（真贡献）；但对手决策变量仍是单纯形 `y`，熵正则罩得住，定理成立（main.tex 第101行：keeps the min–max **form**）。

### 8.3 加「会训练的生成器 teacher」破坏的是**哪根支柱**

加 PAIRED 式生成器 teacher，level 不再来自固定 buffer `Λ_n`，而由 teacher 参数 `φ` 生成。这**同时打掉两根支柱**：

1. **有限固定 buffer**：定理全部数学依赖「`y` 是固定有限集 `Λ_n` 上的单纯形点」——`y*=softmax(s/α)` 是 **n 维** softmax；对偶 LSE 的 `∇²Z=diag(p)−ppᵀ⪯I` 全局 1-光滑（§7.9 承重步骤）；`Φ_α` 的 `α log n` 光滑界里那个 **`n` 就是 buffer 大小**。teacher 生成 level → `y` 支撑集随 `φ` 连续变化、不再定维 → §7.9 那套对偶全局光滑论证**直接失效**。
2. **熵正则只罩 `y`、罩不住 `φ`**：现对手 = 选 `y`（线性进入 + 熵 → α-强凹 ✅）；加 agent 后对手要 `max_φ` 一个 regret 目标，**`φ` 是神经网络、非凹**，`αH(y)` 给不了 `φ` 强凹（main.tex 第205行 Crucial structural point 的反面）。

→ **净判**：破的**不是 general-sum**（§8.1 已有），而是「**对手决策变量是单纯形 `y`**」这根支柱。一旦对手跳到非凸 `φ`，退回 NCC 明说无解的 non-zero-sum open problem（§279 末：NCC leaves the non-zero-sum case as open）。这就是 memory `auction-mechanism-is-plr-not-paired` 「载体必须是 PLR」背后的**真正理由 = 收敛性**。

### 8.4 若后续仍要加（作者计划：性能不足时再引入）—— 唯一保住定理的路线

| 加 agent 的方式 | 定理 | 代价 |
|---|---|---|
| **分层**：teacher agent 只**生成候选 level 填进 buffer**，estimator+auction 仍在**固定 buffer** 上做课程（类 ACCEL edit/replay 分离） | ✅ **保住**（定理罩「给定 buffer 的课程求解」层） | teacher 生成层本身无收敛保证（但 ACCEL/PLR 也都没有，可接受） |
| **彻底对抗**：teacher↔student 真 `min_x max_φ`（PAIRED 式） | ❌ 作废 | 退回无保证的 non-zero-sum open problem，丢掉本篇最硬的理论卖点 |

→ **建议**：若将来加，优先「**分层**」（agent 刷 buffer + 现有 auction 在固定 buffer 上跑）——既得生成器表达力，又保住已证定理；**勿**直接 PAIRED 式对抗。
